# What is Poly-Predict?

Poly-Predict is a virtual prediction market platform where users compete using virtual credits on real-world events sourced from [Polymarket](https://polymarket.com). No real money is involved â€” it's a risk-free way to test your forecasting skills against other players.

New users receive **10,000 credits** to start placing bets on live prediction markets. Events, odds, and outcomes are synced directly from Polymarket, so you're always trading on real data.
